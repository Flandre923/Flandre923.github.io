关于最近几日不知道做些什么。
2021-04-24

关于最近几日不知道做些什么。很闲，不知道做什么，又不想打游戏，没什么好打额。主要笔记本玩不了~~~

开学了好几天，基础课学的差不多了，线代也马上要考试了，今天就结课了。要准备考试，虽然没怎么认真听课，但是觉得还是会一点吧，复习复习应该可以过，值得吐槽的就是最近的忘记了英语和毛概的作业导致有几节没有写，github上刷课的一个浏览器插件挺好用的，搜的题挺全的，[小公举][https://github.com/CodFrm/cxmooc-tools]，还有就是油猴的U校园插件也挺好用，不用自己去一片一篇搜索答案了，虽然作业搜不到，就是了，不得不吐槽一下作业的不少啊。而且难做。英语渣真是累。看一些别人的项目也真的累。又要吐槽了，

上面都是不是重点啦，不是这次写这个博客的目的，因为，开学到现在也不知道自己做了什么，最近学了一些爬虫，可是到自己写时候无从下手，也不知道要做些什么好，没人给点建议，或者是给点任务，然后，想自己鞋垫什么也找不到目标，也找不到自己想要做什么，如果真的说差什么的话，那大概是差钱吧，（嘛，，开个玩笑，）虽然真的差钱就是了，不过，想改变这种情况努力学着，但是发现并不是自己在学校就能凭借这些改变的，不如直接去做一些闲工来的更加快就是了，不过这样做的了就觉得自己学的东西是不是毫无意义？努力就是不是什么都做不了？觉得自己就不能生产点什么出来吗？反思自己学了什么 java spring mybatis python 最近学的爬虫，等等吧，都好像拥不上啊。是我自己太缺乏想象力了。还是自己没有总结反思了？还是自己没什么天分了，（啊啊，唯独不想承认这点），如果说最近想做什么的话，那还真是找不到，上午看了flask，但是有什么呢？没找到什么用的地方，或许可做一个网站，开一个项目，那要做一个什么网站了？做出来分享出去或许是一个不错的注意。那应该从做些什么东西开始，感觉自己一个人做一个网站还是项目有点大，而且去敲一些什么商城项目又觉得没什么意思，不如做一些对自己有用的东西，但是自己又想出来做什么，B站发的视频有几期了，还是有一些播放量的， 颇为欣慰，是不是应该去完善哪个个项目？自己去做更多的东西了？已经好久没动过哪个项目了。该不该继续做下去？还是说？因为服务器到期了，我也没有继续续下去的动力了，因为本来经济就不富裕，不想在增加负担了。

不过学习的还是能想到做什么的，因为还有好多课没学，也有要考试的了。比如模电和线代，一个是不会，一个是怎么学要考试了。该做什么，甚是迷茫啊。如果有个一人和我一起想做点什么就好了。还是想不到什么创意。或者说好玩的来做一下
不过还是先学习去吧，先把自己的专业课搞好吧。然后再说其他的不过，专业课让自己提前学的画又提不起来动力，还是喜欢听一遍老师讲的到最后在学习学习？那平常闲下来的时间做些什么了？
还是希望找到一些自己好玩的吧。最近在玩雀魂，不如想象这方面能做点什么？我想做一个当前胡牌的点数大小，好确定是不是要虎牌了。不过这方面我也没有知识，我应该怎么去做了、我应该用什么语言？我应该去用那些工具，我怎么读取到我当前的牌型，我怎么判断我的手牌，我怎么写逻辑去判断当前的点数？

很多不知道的东西，也没人能告诉我应该去怎么做，不知道python能不能做到，能的话我应该去怎么做，用什么库，去做哪些事情？

后记；
